Operatore ternario: aiuta a semplificare una IF
        
	1 condizione
	2 cosa fare se vero
	3 cosa fare se falso

int a=10;
        int b=20;
        
        int max;
        
        if(a>b) {
            max=a;
        }else {
            max=b;
        }
        
        
        max=(a>b)?a:b; ------> ?= if : = else
